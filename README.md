# Cristian Costoiu Portfolio — Apple Clean Next Level

Static portfolio website ready for Netlify.

## Run locally

Just open `index.html` in a browser.

## Deploy on Netlify

1. Unzip the folder
2. Drag the whole folder into Netlify Deploy manually / Add deploy
3. The site updates on the same link if you deploy inside the same project

## Files

- `index.html` — page structure
- `styles.css` — full design
- `script.js` — animations and interactions
- `data.js` — all CV content in one place
- `assets/Cristian_Costoiu_CV.pdf` — updated CV
